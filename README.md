# 555Timer-Sequential-Digital-Clock

A hardware-based digital clock built entirely on breadboards using a 555 timer, sequential logic gates, and 7-segment displays. This Digital Logic Design (DLD) project implements a fully functional digital clock with manual hour and minute set features utilizing purely discrete hardware components.

**Author:** Muhammad Shayan Shahbaz  
**Simulation Software:** Proteus  

---

## Hardware Bill of Materials (BOM)

The physical breadboard implementation was constructed using the following components:

* **Clock Generation:** 555 Timer IC
* **Sequential Logic & Decoding:** CD4026 IC (Decade Counter & 7-Segment Decoder)
* **Combinational Logic:** 3-Input AND Gates (e.g., 74LS11)
* **Displays:** Common Anode 7-Segment Displays
* **Diodes:** Zener Diodes & Standard Signal/Rectifier Diodes
* **Passive Components:** Resistors and Capacitors 
* **Input & Power:** Push Buttons (for time-setting overrides) & Battery Power Supply
